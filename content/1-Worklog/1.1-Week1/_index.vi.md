### Kết quả đạt được trong Tuần 1:

* Hoàn thành quá trình làm quen với chương trình thực tập First Cloud AI Journey và nắm rõ quy trình làm việc của nhóm.

* Nắm được các kiến thức nền tảng về AWS Cloud, AWS Global Infrastructure, Regions, Availability Zones và AWS Well-Architected Framework.

* Hoàn thành việc phân tích yêu cầu của dự án Smart Attendance SaaS, xác định được các yêu cầu chức năng và phi chức năng của hệ thống.

* Tìm hiểu cấu trúc ban đầu của dự án và định hướng phát triển kiến trúc AWS Serverless cho hệ thống.

* Xác định các dịch vụ AWS cốt lõi sẽ được sử dụng trong giải pháp, bao gồm:

  * Amazon Cognito
  * Amazon API Gateway
  * AWS Lambda
  * Amazon DynamoDB
  * Amazon S3
  * Amazon CloudFront

* So sánh mô hình Three-Tier truyền thống với AWS Serverless và lựa chọn kiến trúc Serverless làm nền tảng triển khai nhờ khả năng mở rộng, tính sẵn sàng cao, bảo mật và tối ưu chi phí.

* Xây dựng nền tảng kiến thức và định hướng thiết kế kiến trúc cho các giai đoạn tiếp theo của dự án Smart Attendance SaaS.

### Mục tiêu Tuần 10:

* Nghiên cứu triển khai hạ tầng bằng Infrastructure as Code (IaC).
* Tìm hiểu quy trình CI/CD cho ứng dụng AWS Serverless.
* Tăng cường bảo mật hệ thống bằng IAM và AWS KMS.
* Tối ưu luồng xử lý Backend cho hệ thống Smart Attendance SaaS.

### Kết quả đạt được Tuần 10:

* Nắm được phương pháp triển khai hạ tầng bằng AWS SAM và AWS CloudFormation.

* Hiểu quy trình CI/CD sử dụng AWS CodePipeline và AWS CodeBuild.

* Thiết lập cơ chế bảo mật bằng AWS IAM và AWS KMS nhằm bảo vệ dữ liệu hệ thống.

* Tối ưu luồng xử lý Backend giữa API Gateway, Lambda và DynamoDB.

* Hoàn thiện cấu hình triển khai và chuẩn bị cho giai đoạn hoàn thiện tài liệu dự án.

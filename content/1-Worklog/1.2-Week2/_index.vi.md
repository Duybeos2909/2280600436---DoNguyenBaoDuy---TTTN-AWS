* Phân tích yêu cầu nghiệp vụ của dự án Smart Attendance SaaS.

* Hiểu vai trò và cách các dịch vụ AWS tương tác trong kiến trúc tổng thể.

### Kết quả đạt được Tuần 2:

* Hoàn thành việc phân tích yêu cầu nghiệp vụ của hệ thống Smart Attendance SaaS.

* Lựa chọn được các dịch vụ AWS cốt lõi phù hợp với kiến trúc Serverless, bao gồm:

  * Amazon Cognito
  * Amazon API Gateway
  * AWS Lambda
  * Amazon DynamoDB
  * Amazon S3
  * Amazon CloudFront
  * Amazon Route 53
  * AWS WAF

* Hiểu rõ vai trò và sự phối hợp của từng dịch vụ AWS trong kiến trúc tổng thể.

* Hoàn thành việc ánh xạ các chức năng nghiệp vụ với các dịch vụ AWS tương ứng.

* Xây dựng phiên bản đầu tiên của kiến trúc AWS Serverless làm cơ sở cho việc thiết kế chi tiết trong các tuần tiếp theo.
